        # [Oracle] 프로시저 커서(CURSOR) 쿼리 확인

        Oracle 프로시저 커서를 출력하기 위한 Mybatis xml 설정과 쿼리 호출 샘플

        # 커서 ( Cursor)

        • SQL 커서는 Oracle 서버에서 할당한 전용 메모리 영역에 대한 포인터.
        • 질의의 결과로 얻어진 여러 행이 저장된 메모리상의 위치.
        • SELECT 문의 결과 집할을 처리하는데 사용

        # Sample(Java)

        Map param = new HashMap();
param.put("param1", param1);
param.put("resultList" , ""	);
		
oracleViewAppMapper.selectList("jw.newsCenter.selectNewsCenterData", param);	
model.addAttribute("articleList", param);  // param에 결과값이 담긴다.

        # Sample(Xml)

        TIP

        mode=OUT 에 선언된 resultName 을 Key 로하여 전달된 파라미터에 결과값이 담긴다.

        <resultMap id="newsCenterMap" type="CamelMap" />
	
	<select id="selectNewsCenterData" statementType="CALLABLE" parameterType="CamelMap">
	{ CALL pkg_tis_portlet.sp_get_notice_info_now(
	    #{param1 , mode=IN},
	    #{resultList, mode=OUT, jdbcType=CURSOR, javaType=ResultSet, resultMap=newsCenterMap} 
	    )
	}
</select>
