            # [CentOS7] NFS를 이용한 NAS 설정(Client)

            NAS 서버가 구축된 상태로 Client 설정

            # 1. 서버의 공유폴더와 mount 시킬 폴더를 생성

            $ mkdir cms
$ pwd
/DATA/cms

            # 2. NFS 서버의 접근가능한 IP/공유폴더 확인

            $ showmount -e 172.22.101.122
Export list for 172.22.101.122:
/data/col1/cms 172.22.101.192

            showmount 명령어 사용할 수 없을경우 (root 계정)

            $ yum install showmount

            # 3. NFS 서버의 공유폴더와 클라이언트 폴더 마운트

            $ mount -t nfs 172.22.101.122:/data/col1/cms /DATA/cms

            # 4. 설정 확인 (이중화구성시)

            $[cms@SVV-CMSWAS-01 cms]$ vi index.html  <-- 1번서버에서 파일 생성
$[cms@SVV-CMSWAS-01 cms]$ ls
index.html

            $[cms@SVV-CMSWAS-02 cms]$ ls   <-- 2번서버에서 동일 파일 확인
index.html

            # 5. 부팅시 자동 마운트 설정

            $ vi /etc/fstab
...

172.22.101.122:/data/col1/cms /DATA/cms nfs hard 0 0
